## SentinelAI Investor Narrative

### One-Line Product

SentinelAI is a cloud-native Security Operating System for enterprises: a multi-tenant platform that combines endpoint telemetry, streaming analytics, AI-driven detection, and operational SOC workflows.

### ICP

- enterprise SOC teams
- MSSPs
- security engineering teams
- mid-market companies graduating from fragmented SIEM + EDR tooling

### Pricing Model

- Free:
  - limited agents
  - limited EPS
  - community support
- Pro:
  - per-agent monthly pricing
  - metered event ingestion
  - standard retention
- Enterprise:
  - custom per-agent contract
  - volume event pricing
  - dedicated tenancy / BYOK / advanced response

### Positioning

- vs CrowdStrike:
  - broader SOC workflow orientation
  - more open streaming architecture
- vs SentinelOne:
  - stronger SaaS control-plane and observability-first story
- vs Datadog Security:
  - endpoint-native EDR plus SOC workflow positioning

### Why Now

- explosion of endpoint telemetry
- cloud-native security stack replacement cycle
- SOC labor shortage
- demand for AI-assisted detection and triage

### YC 10-Slide Deck

1. Problem: fragmented security operations and alert overload
2. Solution: unified Security Operating System
3. Architecture: agent -> kafka -> ML -> control plane -> response
4. Why now: AI + telemetry explosion + cloud migration
5. Product demo: live host telemetry to incident to response approval
6. Market size: endpoint security + SIEM + MDR software budgets
7. Business model: per-agent + event ingestion + enterprise modules
8. Competition: CrowdStrike, SentinelOne, Datadog Security, Microsoft Sentinel
9. Traction: prototype platform, multi-tenant architecture, cloud-ready deployment
10. Vision: default operating layer for enterprise cyber defense
